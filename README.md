RKPR
====

How to setup the RKPR Development Environment
---------------------------------------------

Clone a copy of the main RKPR repo

```bash
git clone https://github.com/jried31/RKPR.git
```

### Setup Node.js Environment 

Enter Node.js directory
```
cd Cloud
```

Install dependencies
```
npm i
```
